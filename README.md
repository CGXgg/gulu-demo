## 开始使用

1.添加 CSS 样式
使用本框架前，请在 CSS 中开启 border=box

    ```
    *,*::before,*::after{box-sizing:border-box}
    ```

    IE 8 及以上浏览器都支持此样式
    你还需要设置默认颜色等变量（后续会改为 SCSS 变量）

    ```
    html {
            --font-size: 14px;
            --button-height: 32px;
            --button-bg: white;
            --button-active-bg: #eee;
            --border-radius: 4px;
            --color: #999;
            --border-color: #999;
            --border-color-hover: #666;
          }
    ```
    IE 15 及以上浏览器都支持此样式
    2. 安装gulu
    ```
    npm i --save frank-test-1-1
    ```
    3.引入 gulu

    npm i -- save gulu
